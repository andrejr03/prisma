# Prisma Engineering Dashboard Prototype

This folder contains a static design prototype for the Prisma engineering dashboard.

It is not connected to the Prisma backend, API, runtime artifacts, local index, or
evaluation runner. It visualizes concepts implemented by Prisma, including
evaluation status, prompt regression, runtime metrics, workflow traces, retrieved
context, and citation grounding.

All data shown in the prototype is mock data.

## Open

Open `index.html` directly in a browser.

No server is required. No build step is required. No dependencies are required.
The prototype works fully offline from the local filesystem.
