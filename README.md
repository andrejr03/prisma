# Prisma
> Production LLM Engineering Platform

Prisma is a local-first engineering platform for building production-grade LLM systems. It combines RAG, agent workflows, evaluation, prompt regression, observability, and CI/CD into a reproducible development workflow.

The MVP does not require Supabase or any hosted database. It is designed to run locally while preserving production-readiness concerns such as traceability, quality gates, and maintainable system boundaries.
